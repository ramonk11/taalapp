# LinguaFlow 🗣️

AI-powered taaltutor — leer een taal door te praten.

## Features
- Chat met een AI-tutor die zich aanpast aan je niveau
- 10 talen beschikbaar (Spaans, Frans, Duits, etc.)
- Niveau A1 t/m C2 (CEFR)
- 4 oefenmodi: vrij gesprek, scenario's, grammatica, woordenschat
- Foutentracking en voortgangsstatistieken
- Streak-systeem voor dagelijks oefenen

## Setup
1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000
4. Vul je Anthropic API key in en begin!
